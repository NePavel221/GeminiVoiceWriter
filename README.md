# Gemini Voice Writer

Десктопное приложение для голосового ввода текста с использованием Google Gemini AI. Нажмите горячую клавишу, продиктуйте текст — и приложение распознает речь и вставит текст в активное поле ввода.

## Возможности

- голосовой ввод в любое приложение
- распознавание речи через Gemini AI
- настраиваемая горячая клавиша
- выбор модели Gemini
- overlay / floating UI во время записи
- сворачивание в трей
- автокопирование и автовставка текста
- история и базовая статистика

## Быстрый старт

### Требования
- Windows
- Python 3.10+
- Gemini API key: https://aistudio.google.com/app/apikey

### Установка

```bash
git clone https://github.com/NePavel221/GeminiVoiceWriter.git
cd GeminiVoiceWriter
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Запуск

```bash
python main_v2.py
```

## Настройка

После первого запуска:
1. вставьте Gemini API key
2. настройте горячую клавишу
3. выберите модель и метод распознавания
4. при желании включите auto-copy / auto-paste / overlay

## Структура проекта

- `main_v2.py` — точка входа
- `core/` — запись, транскрибация, хоткеи, вставка текста
- `ui/` — окна, overlay, виджеты
- `services/` — AI-провайдеры
- `utils/` — конфиг, история, пути, логирование
- `assets/` — звуки и прочие ресурсы

## Сборка

```bash
pip install pyinstaller
pyinstaller GeminiVoiceWriter.spec
```

Собранный файл появится в `dist/`.
